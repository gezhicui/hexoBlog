axios

字数统计: 982 | 阅读时长 ≈ 4

axios(官网: https://github.com/axios/axios )是一个基于Promise用于浏览器和node.js的HTTP客户端。
它具有以下特征:

  • 支持浏览器和node.js心
  • 支持promise
  • 能拦截请求和响应
  • 自动转换JSON数据

安装

npm install axios -S

axios的基本用法

axios常用API

  • get :查询数据
  • post:添加数据
  • put :修改数据
  • delete:删除数据
